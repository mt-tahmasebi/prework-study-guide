HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The head elements start from 1 to 6 and can be used only once
  • The body area can be divided into different sectinos

CSS

  • A margin is the space around an element
  • A padding is the space around the content within an element
  • Style indication can be associated with a range of elements of specific ones
  • Margin, padding, colour, and other style related attributes can be associated with elements
  • An easy way of associating a style with many elements is the use of class attribute
